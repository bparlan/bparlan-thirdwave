# Terminal UI Module

The core of the dashboard's visual interface.

## Modules
- `panels/`: Contains `LeftSidebar.tsx` and `MainContent.tsx`.
- `Header.tsx`: Title bar and workspace status indicator.
- `Footer.tsx`: External links and system uptime display.
- `ThemeUtils.ts`: Handles accent color mapping per dashboard tab.
- `Types.ts`: Defines interface contracts for theme and wallet data.
